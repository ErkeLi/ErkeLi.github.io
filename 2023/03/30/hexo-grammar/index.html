hexo语法

介绍了使用hexo构建页面的一些基础操作

页面的生成

hexo g 完整命令为hexo generate，用于生成静态文件

hexo s 完整命令为hexo server ， 打开浏览器输入 localhost:4000，可以预览效果

hexo d 完整命令为hexo deploy，用于将本地文件发布到github等git仓库上

hexo new [主题名] 网站名 创建新的页面，主题可以为空。
hexo n hexo_grammar
